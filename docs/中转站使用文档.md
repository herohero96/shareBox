# MiniMax AI 中转站使用指南

> 基于 New API 搭建的 AI 模型代理网关，统一提供 OpenAI 兼容接口。

---

## 📡 服务地址

```
Base URL: http://51.21.252.41:3000/v1
```

> ⚠️ 当前仅供测试，如需 HTTPS 请联系管理员。

---

## 🔑 获取 API Key

1. 打开注册链接：**[注册新账号](http://51.21.252.41:3000/register)**
2. 注册并登录后，进入后台 → **令牌** → **添加令牌**
3. 设置令牌名称、额度（或勾选无限额度）、可用模型
4. 创建后复制完整的 `sk-` 开头的 API Key

> 如已有账号，直接登录后在 **令牌** 页面创建即可。

## 🚀 快速使用

### 1. 在第三方客户端使用（推荐）

| 客户端 | 配置方式 |
|--------|---------|
| **OpenCat** | 添加自定义 Provider → Base URL: `http://51.21.252.41:3000/v1` |
| **Cherry Studio** | 选择 OpenAI 兼容 → 填入地址和 Key |
| **LobeChat** | 自定义 OpenAI API → 填入地址和 Key |
| **NextChat** | 自定义接口 → 填入地址和 Key |
| **Ama 问天** | 自定义 Provider → 填入地址和 Key |

### 2. 直接 API 调用

```bash
curl http://51.21.252.41:3000/v1/chat/completions \
  -H "Authorization: Bearer sk-你的密钥" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "MiniMax-M2.7",
    "messages": [
      {"role": "user", "content": "你好"}
    ]
  }'
```

### 3. 使用 OpenAI SDK

**Python:**

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://51.21.252.41:3000/v1",
    api_key="sk-你的密钥"
)

response = client.chat.completions.create(
    model="MiniMax-M2.7",
    messages=[{"role": "user", "content": "你好"}]
)
print(response.choices[0].message.content)
```

**Node.js:**

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://51.21.252.41:3000/v1',
  apiKey: 'sk-你的密钥'
});

const response = await client.chat.completions.create({
  model: 'MiniMax-M2.7',
  messages: [{ role: 'user', content: '你好' }]
});
console.log(response.choices[0].message.content);
```

## 🔧 常见问题

**Q: 返回 `Invalid token`？**
A: 检查 API Key 是否复制完整（`sk-` 开头），以及是否在后台创建了令牌。

**Q: 提示 `model_price_error`？**
A: 当前为自用模式，如遇到请联系管理员。

**Q: 响应慢？**
A: MiniMax 官方模型约 60 tps 输出速度，高速版约 100 tps。

**Q: 如何查看调用记录？**
A: 管理员在后台 → **日志** 页面查看所有请求记录。
